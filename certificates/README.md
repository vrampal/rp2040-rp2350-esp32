## Certificates for differents usages

Files "demo-selfsigned-*.pem" and "demo-selfsigned-*.key" are self-signed certificates for HTTPS server demo.

Do NOT use in production as the key is public.

Files "isrg-*.pem" are root certificates from Internet Security Research Group / Let's Encrypt.

They have been retreived from [https://letsencrypt.org/certificates/](https://letsencrypt.org/certificates/).

Files "gts-*.pem" and "globalsign-root-*.pem" are root certificates from Google Trust Services.

They have been retreived from [https://pki.goog/repository/](https://pki.goog/repository/).

Files "sectigo-*.pem" are root certificates from Sectigo.

They have been retreived from [https://www.sectigo.com/knowledge-base/detail/Sectigo-Root-Certificates](https://www.sectigo.com/knowledge-base/detail/Sectigo-Root-Certificates).
